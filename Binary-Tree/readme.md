## recursion
在递归中， 若求解过程需涉及到所有点（所有点地位相同），那么在辅助函数中遍历当前点的左右子树， 在最终求解时， 需将根节点在辅助函数中遍历，同时将左右子节点作为根带入主函数，得到的结果才是所有点均遍历的结果
